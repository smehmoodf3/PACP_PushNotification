ECLLogic.DataReader Class
This class reads contact data from the data source.

Access: Project
Base Classes: Object
  Members Description  
    objConn OleDb Connection to the data source

 
    log Define a static logger variable so that it references the Logger instance named "DataReader".

 
    GroupList This method extracts the group names from the data source. Returns them in a vector of strings.

 
    DataReader Default constructor.

 
    openDB Opens an ODBC connection

 
    closeDB Close the ODBC connection

 
    getContactList Extracts the names of all people in the indicated group. Returns them as a vector of strings.

 
    getContactData Extracts the contact details of the named person. The resulting string array contains an entry for each non-empty cell in the person's row, formatted according to the rules from the properties file.
