ECLLogic.ECLConfiguration Struct
This structure is used to send Connection informaton. It has twelve variables.

Access: Public
  Members Description  
    recipientFile File containing user information(Server Port Email/Pin)

 
    PushTo Push Destination(Channel/Catcher)

 
    PushType has a value 0(Browser-Channel) or 1(Browser-Channel-Delete)

 
    ConnectionType This can be Access,Excel,MSSQL

 
    Database Database name,used in case of MS SQL

 
    File_Server_Name File name(.xls or .mdb) or Server Name(used in MSSQL)

 
    Sheet_Table_Name Worksheet Name(Used in Excel) or Table name(used in Access or MSSQL)

 
    ColumnFormat The format in which data is displayed on the handheld

 
    fileStoreLoc The location where the html file should stored after it has been formed

 
    WebRoot The http:\\ URL where the read and unread icon files are

 
    ChannelName The name to be displayed as Channel Title on the handheld

 
    Source The Source of the input(GUI/Console/Service)
